# خارطة الطريق — أثر

الفلسفة: نسخة لينة تطلق بسرعة وتثبت الجودة، ثم نعمّق "الذكاء الكامل" تدريجياً. نقاوم إغراء بناء كل شي.

## المرحلة 1 — الـ MVP

**الهدف:** إثبات أن "أثر" ينتج محتوى عربي *موثّق وعلى لسان العلامة* يدفع له الـSMB ويبقى.

**المزايا (الحد الأدنى — لا شيء غيرها):**
- التهيئة الذكية / دماغ الشركة (سحب من الموقع/الحسابات + أسئلة قصيرة → ملف معرفة).
- محرّك بحث → بوست بمصادر (يعرض المصدر تحت كل معلومة).
- كتابة عربية بنبرة العلامة (لينكدإن + X بصياغة مختلفة).
- توليد صور بنص عربي صحيح (gpt-image يولّد النص العربي + تحقّق بصري + overlay احتياط).
- تقويم محتوى سعودي + تذكيرات.
- **نشر يدوي مُيسّر** (نسخ بضغطة + فتح المنصة + تذكير مجدول).
- سير اعتماد بسيط (الذكاء يجهّز، العميل يعتمد/يعدّل).
- اشتراك + دفع Moyasar.

**مهمة بناء #١:** اختبار ٢٠ صورة لقياس نسبة تكسّر العربي قبل اعتماد المعمارية.

**مدة تقديرية:** ~٣ أشهر لفريق صغير لإطلاق صلب (تعتمد على تركيز الانتباه).

## المرحلة 2 — V1 (بعد تعلّم الإطلاق)

- **النشر التلقائي المتوافق** (لينكدإن + X عبر APIs رسمية) — وضع autopilot، يصير مبرّر ترقية وسعر أعلى.
- **التحليلات** (أداء البوستات + تقرير شهري) — تُفتح مع وصول الـAPI.
- تحسين جودة المحرّك بناءً على ملاحظات أول دفعة.
- باقة ثانية (لايت أو أعلى) حسب ما يكشفه السوق.
- اختبار نموذج عربي-أصيل (Jais/قلم) عبر الـContentProvider لخفض التكلفة/تحسين اللهجة.

## المرحلة 3 — التوسّع

- وكالات / متعدد العلامات + **White-label** (نقطة قوتك).
- باب الأفراد self-serve (تحت براند فرعي محتمل).
- منصات إضافية (إنستغرام / تيك توك) + فيديو.
- حلقة تعلّم ذاتي (الذكاء يحسّن المحتوى من التحليلات) — تعميق "الذكاء الكامل".
- توسّع خليجي (إمارات/قطر).
- تكامل مع منتجاتك الأخرى (مثل أدير) كحزمة.

## ما تم تأجيله عمداً (Not Now)

- النشر التلقائي والتحليلات (V2 — تحتاج وصول API).
- تعدّد العملاء/العلامات و White-label و مقاعد فريق متقدمة.
- ردود تلقائية على التعليقات/الرسائل (**منطقة أتمتة خطرة — تجنّبها مبدئياً**).
- منصات غير لينكدإن/X، والفيديو.
- باب الأفراد، التوسّع الخليجي، حلقة التعلّم الذاتي.

> تسمية ما **لا** نبنيه الآن لا تقلّ أهمية عمّا نبنيه — هي درعك ضد تضخّم النطاق وتشتّت الفريق.
