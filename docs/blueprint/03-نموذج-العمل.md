# نموذج العمل — أثر

## مخطط نموذج العمل (Business Model Canvas)

- **شرائح العملاء:** الشركات الصغيرة والمتوسطة السعودية النشطة على لينكدإن/تويتر بلا فريق محتوى (متاجر، مطاعم، عيادات، خدمات مهنية، علامات شخصية لمؤسّسين). *(الأفراد والوكالات = شرائح مؤجّلة.)*
- **القيمة المقترحة:** محتوى عربي احترافي مبنيّ على معلومة موثّقة وبنبرة العلامة، بجزء بسيط من سعر الوكالة وبسرعة الآلة، مع تهيئة عميقة تعرف الشركة.
- **القنوات:** Google Ads (نية عالية)، لينكدإن/X العضوي (المنتج يسوّق نفسه)، شبكة المؤسّس، شراكات (حاضنات، غرف تجارة، مزوّدي خدمات SMB)، محتوى تعليمي.
- **العلاقة مع العملاء:** self-serve مع تهيئة موجّهة (onboarding)، دعم بالعربي، نجاح عملاء خفيف في البداية (الـSMB يحتاج طمأنة).
- **مصادر الإيراد:** اشتراك شهري/سنوي (باقة واحدة عند الإطلاق ~٥٩٩ ريال). لاحقاً: باقة وكالة، باقة لايت، إضافات استهلاك.
- **الموارد الرئيسية:** المحرّك (التهيئة + بحث→بوست + نبرة + صور)، Claude API، قاعدة معرفة لكل عميل، الفريق (أنت + علي).
- **الأنشطة الرئيسية:** تطوير وتحسين جودة المحتوى العربي، prompt/agent engineering، اكتساب وبيع، نجاح عملاء.
- **الشركاء الرئيسيون:** Anthropic (Claude)، Moyasar (مدفوعات)، مزوّد توليد صور، لاحقاً منصات النشر (لينكدإن/X) عند تفعيل الأتمتة، ومزوّد نموذج عربي محتمل (Jais/قلم).
- **هيكل التكاليف:** Claude API (التكلفة المتغيّرة الأكبر)، توليد الصور، استضافة، Moyasar (~٢.٧٥٪+ للمعاملة)، تسويق (CAC)، وقت التطوير. *(لا تكاليف API نشر في المرحلة الأولى — ميزة هامش كبيرة.)*

## استراتيجية التسعير

**باقة واحدة عند الإطلاق — "أعمال" ~٥٩٩ ريال/شهر** (نطاق اختبار ٤٩٩–٧٩٩)، خصم سنوي ~شهرين مجاناً، تجربة مجانية ٧ أيام. **بدون باقة مجانية دائمة فيها نشر/توليد مكثّف** (تحمي التكلفة).

**المنطق (تسعير بالقيمة لا بالتكلفة):**
- المرجع هو الوكالة السعودية (٣,٠٠٠–٧,٠٠٠ ريال)، لا الأدوات العالمية (٣٩ دولار). عند ٥٩٩ ريال أنت ~عُشر سعر الوكالة → قصة ROI قاتلة ("محتواك بـ٥٩٩ بدل ٣,٥٠٠").
- فوق أدوات الـ١٠–٢٥ دولار بوضوح، فما تدخل حرب أسعار، وتحمي مصداقيتك B2B (٢٩٩ ريال يضرّ التموضع كبديل وكالة).
- **السقف:** لا ترفع لمستوى الوكالة — أنت تستبدل عمالة المحتوى، لا الاستراتيجية/العلاقة/التوزيع.
- تكتيك إطلاق: ابدأ عند الطرف الأدنى (~٤٩٩–٥٩٩) لتجميع عملاء وشهادات بسرعة، وارفع مع إثبات القيمة (وخصوصاً لمّا يجي النشر التلقائي في V2 كمبرّر ترقية).

**الباقات المؤجّلة (خارطة):** "أعمال لايت" أرخص للأنشطة الصغيرة جداً · "وكالة/متعدد العلامات" + White-label · باب الأفراد self-serve (~١٠–٢٥ دولار، تحت براند فرعي محتمل).

## اقتصاديات الوحدة (تقديرية — تحتاج تحقق)

افتراضات: عميل ينتج ~١٠٠ مسودة و~٦٠ صورة شهرياً، نشر يدوي (بلا تكلفة API نشر).

| البند | تقدير شهري لكل عميل |
|---|---|
| توليد المحتوى (Claude، بحث+كتابة عربي) | ~١٠–٢٥ دولار |
| توليد الصور | ~٢.٥–٥ دولار |
| Moyasar (على ٥٩٩ ريال) | ~٤–٥ دولار |
| **إجمالي تكلفة متغيّرة** | **~١٧–٣٥ دولار (~٦٥–١٣٠ ريال)** |
| **الإيراد** | **~٥٩٩ ريال (~١٦٠ دولار)** |
| **هامش إجمالي تقديري** | **~٧٥–٨٥٪** |

- **CAC (تقديري):** ٣٠٠–٨٠٠ ريال في البداية (Google/لينكدإن + شبكة + بيع مباشر). يحتاج قياس فعلي.
- **LTV:** عند ٥٩٩ ريال وهامش ~٨٠٪ واحتفاظ ١٠ أشهر (افتراض متحفظ) ≈ **~٤,٨٠٠ ريال** قيمة إجمالية لكل عميل. نسبة LTV/CAC ممتازة *إن* تم ضبط الـchurn.
- **فترة الاسترداد:** غالباً <٢ شهر إن بقي CAC منطقياً.
- **الخطر الأهم على الاقتصاد:** churn (الـSMB يلغي بسرعة لو ما شاف قيمة) + تكلفة Claude بالعربي إن زاد الاستخدام. السقوف وجودة المحتوى يحميان الاثنين.

## مسار الإيراد المتوقع (سيناريو متحفظ — ١٢ شهر)

بافتراض إطلاق بعد تحقّق، وبيع بطيء واقعي:

| الفترة | عملاء مدفوعون (تراكمي) | ARR تقريبي |
|---|---|---|
| شهر ٣ (إطلاق) | ١٠–٢٠ (روّاد + شهادات) | ~٧٢–١٤٤ ألف ريال |
| شهر ٦ | ٤٠–٦٠ | ~٢٨٨–٤٣٢ ألف ريال |
| شهر ١٢ | ١٠٠–١٥٠ | ~٧٢٠ ألف – ١.١ مليون ريال |

أرقام محافظة عمداً (لا hockey-stick). تعتمد كلياً على ضبط الـchurn وإثبات الجودة. ارفع الأسعار تدريجياً لتحسين ARR لكل عميل بدل مطاردة الحجم فقط.
