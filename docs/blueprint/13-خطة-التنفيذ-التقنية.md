**مؤجّل (V2+):** النشر التلقائي (LinkedIn/X API)، التحليلات، الردود التلقائية، الوكالة/White-label.

## البيئات
- **dev:** تطوير محلي (Docker compose: API + Postgres + Redis + MinIO).
- **staging:** نسخة قبل الإنتاج لاختبار التدفّقات والدفع (Moyasar test).
- **production:** خادم/منطقة مناسبة لإقامة البيانات؛ مفاتيح حيّة.

## CI/CD والجودة
- Pipeline: lint + type-check + tests عند كل PR، ونشر آلي لـstaging.
- اختبارات: وحدة لمنطق المحرّك (خصوصاً اقتران المصدر بالبوست، وتحقّق نص الصورة من gpt-image عبر OCR/رؤية + سلامة العربي)، وe2e للتدفّقات الحرجة (تهيئة → توليد → اعتماد → تصدير).
- مراجعة كود + ترقية البيئات يدوياً إلى production.
- **اختبار جودة خاص:** عيّنة تقييم بشري دورية لمخرجات العربي (هل النبرة/المصدر/الصورة بالمستوى؟) — الجودة هي المنتج.

## بذرة CLAUDE.md للمستودع

```markdown
# أثر — CLAUDE.md

## ما هو
منصّة SaaS عربية تنتج محتوى سوشيال ميديا (LinkedIn/X) للـSMB السعودي:
تهيئة عميقة لكل شركة (BrandProfile) → محرّك بحث→بوست بمصادر → كتابة عربية بنبرة العلامة → صور بنص عربي → تقويم سعودي. النشر **يدوي مُيسّر** في هذه النسخة.

## Stack
- Frontend: Next.js (React), RTL، IBM Plex Sans Arabic، موبايل-أول
- Backend: NestJS (Node/TS)
- DB: PostgreSQL + Prisma
- Jobs: BullMQ + Redis  ·  Storage: MinIO/S3
- AI (نص): Claude API خلف `ContentProvider` interface (لا تستدعِ المزوّد مباشرة من أي مكان آخر)
- AI (صور): OpenAI gpt-image كمزوّد صور ثانٍ جنب Claude
- Payments: Moyasar  ·  Infra: Docker

## مبادئ
- عربي/RTL أولاً في كل واجهة ومخرج.
- **كل معلومة فعلية لازم تُقرن بمصدر (SourceCitation)** — لا اختلاق.
- **البحث حيّ مقيّد بمصادر موثوقة داخل محاور العميل المخزّنة في `BrandProfile`، لا RAG.**
- **لا نشر تلقائي في هذه النسخة** — اعتماد بشري إلزامي قبل النشر اليدوي.
- النص العربي على الصور: gpt-image يولّد النص مدموجاً + **تحقّق بصري إلزامي قبل العرض** (إعادة توليد عند الفشل) + overlay احتياطي.
- تتبّع UsageRecord لكل tenant (مراقبة الهامش) + طبّق سقوف الباقة.
- مرّ كل توليد عبر `ContentProvider` ليسهل تبديل/اختبار نموذج عربي لاحقاً.

## Do / Don't
- DO: عزل بيانات كل tenant، تشفير الأسرار، توافق PDPL.
- DO: توليد "خطة شهر" كـasync job مع تتبّع تقدّم.
- DON'T: مصادقة كوكيز لمنصات النشر (سبب حظر حسابات) — APIs رسمية فقط عند تفعيل النشر لاحقاً.
- DON'T: توسيع النطاق لمزايا V2 (تحليلات/أتمتة/وكالة) قبل إثبات الـMVP.

## خارج النطاق الآن
نشر تلقائي، تحليلات، ردود تلقائية، تعدّد علامات/White-label، منصات غير LinkedIn/X، فيديو.
```
