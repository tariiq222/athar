# أثر — دراسة المشروع (Blueprint)
### «من المعرفة إلى الأثر»

دراسة كاملة لمنصّة **أثر**: محرّك محتوى سوشيال ميديا عربي ذكي للشركات الصغيرة والمتوسطة السعودية (لينكدإن + تويتر). الوعد: نريحك من همّ المحتوى ونكون يدك اليمنى — محتوى مبنيّ على معرفة، يترك أثراً.
> الاسم "أثر" مختار، لكن تأكّد من توفّر الدومين (مثل athar.sa) وسجّل العلامة في SAIP قبل بناء الهوية. الدراسة مبنية على جلسة اكتشاف + بحث سوق فعلي (يونيو ٢٠٢٦). الأرقام المعلّمة "تقديرية" تحتاج تحقق ميداني.

**قرارات الإطلاق المثبّتة:** **بناء-أول** (نبني الـPRD كاملاً مباشرة بلا مرحلة تحقّق مسبقة؛ معايير التحقق صارت بوابة نجاح الدفعة الأولى بعد الإطلاق) · سوق سعودي أولاً · شريحة SMB فقط (الأفراد/الوكالات مؤجّلة) · باقة واحدة ~٥٩٩ ريال · **نشر يدوي مُيسّر** (الأتمتة والتحليلات في V2) · محرّك على Claude خلف `ContentProvider` · **صور gpt-image (OpenAI)** تولّد النص العربي + تحقّق بصري + overlay احتياط · **بحث** = محاور العميل (setup) + بحث حيّ مقيّد بمصادر موثوقة (لا RAG) · **multi-tenant منطقي**.

---

## ترتيب القراءة المقترح
1. ابدأ بـ **الملخص التنفيذي** (القراءة الصريحة للجدوى أهم جزء).
2. ثم **تحليل المنافسين** (الفجوة والتموضع).
3. للقرار التجاري: السوق ← نموذج العمل ← الجدوى/المخاطر ← خارطة الطريق.
4. للبناء: PRD ← قصص المستخدم ← البنية التقنية ← خطة التنفيذ.
5. للتسويق: الشخصيات ← GTM ← القنوات ← المحتوى ← KPIs.

## الخلاصة في سطر
فكرة قابلة للحياة بتموضع ذكي في فراغ حقيقي، وقرار النشر اليدوي خفّض المخاطرة كثير — والقرار المثبّت **بناء-أول**: نبني الـPRD كاملاً مباشرة، ونثبت أن جودتك العربية فوق أدوات الـ١٠ دولار وأن الـSMB يدفع ~٥٩٩ عبر **بوابة نجاح الدفعة الأولى بعد الإطلاق** (تجربة مدفوعة حقيقية، عيّنة مقابل محتواهم الحالي + ChatGPT خام، ≥٣ يدفعون + تجديد الشهر الثاني).
